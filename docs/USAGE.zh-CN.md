# 使用文档

[English](USAGE.md) · **中文**

> Sprite Cutout 有两个标签页 —— **✂️ 裁透明边** 和 **🔲 9-slice**,用顶部标签切换。

---

## 0. 打开

下载仓库(或单独下 `index.html`),**双击 `index.html`** 用浏览器打开。推荐用 Chrome / Edge(「选文件夹」批量功能需要)。无需安装,全部本地运行。

---

## 1. ✂️ 裁透明边

去掉精灵四周的透明留白,让图片紧贴内容边缘。

### 三种喂图方式
| 方式 | 结果 |
|---|---|
| **选文件夹** | 选一个文件夹,裁掉里面每张图,结果写进 `_trimmed` 子文件夹,不动原图。(仅 Chrome/Edge) |
| **选图片** | 选一张或多张图,裁好逐个下载。 |
| **拖拽** | 把图片拖到虚线区,效果同「选图片」。 |

### 选项
- **留边** —— 裁完后四周保留 N 像素透明边。
- **阈值** —— alpha ≤ 此值算透明;调大(如 10)可连半透明羽化边一起裁。

### 自动跳过
全透明的图、没有透明通道的图、已经紧贴边缘的图。

### 结果列表
每行显示缩略图 + `原尺寸 → 裁后尺寸`(或跳过原因)。

## 2. 🔲 9-slice 九宫格

做拉伸不变形的边框 / 面板(圆角、描边不会被拉糊)。

### 步骤
1. **载入图片** —— 「载入图片…」或把图拖进窗口;打开就有示例图。
2. **设分割线** —— 拖动四条金线,或直接填 左/右/上/下 边距像素。
3. **看拉伸** —— 拖目标宽/高滑块,预览区显示九宫格拉伸效果。
4. **导出**:
   - **复制边距数值** —— 一键拿到 Unity / CSS 格式的四个数。
   - **导出九块 PNG** —— 给需要九张图的引擎(`tl/tc/tr/ml/mc/mr/bl/bc/br`)。

### 边距怎么填进引擎
- **Unity** Sprite Border:`L, B, R, T`(左下右上)
- **CSS**:`border-image-slice: T R B L` + `border-width`
- **Cocos scale9 / Laya**:四个 inset 数值直接填

---

## 3. 说明

- **隐私** —— 全部本地运行,图片绝不上传。
- **「选文件夹」依赖 Chrome/Edge** —— 用的是 File System Access API;其它浏览器请用「选图片」/ 拖拽。
- **格式** —— 支持带透明通道的 PNG / WebP / TGA / TIFF。
